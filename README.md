 Sistema de Carrinho de Compras - Mercado do Zé

Um sistema simples de carrinho de compras desenvolvido em Python, simulando um mercado com funcionalidades básicas de compra.

Descrição

Este é um sistema de linha de comando que simula um carrinho de compras de um mercado. O usuário pode adicionar itens, visualizar o carrinho, remover itens, calcular o total e finalizar a compra. O sistema inclui validações de entrada e limites de itens.

Funcionalidades

- **Adicionar itens** ao carrinho com nome e preço
- **Visualizar carrinho** com lista de itens e total
- **Remover itens** específicos do carrinho
- **Calcular total** da compra automaticamente
- **Finalizar compra** com confirmação
- **Limite de itens** configurável (padrão: 5 itens)
- **Validações** de entrada e tratamento de erros
- **Confirmação** ao sair com itens no carrinho
